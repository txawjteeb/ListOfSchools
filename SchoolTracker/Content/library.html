The District’s O3 framework helps to streamline the development of web applications, pages, and web content. By utilizing this framework you effectively extend the District's living style guide and help create a cohesive user experience between the District's products.

Download ZIP

Dowloand and/or Modify

Download the zip of the latest version above.

<link rel="stylesheet" href="path/to/03/dist/library.min.css">

In the head of your html, reference the location to your library.min.css.

This distribution version uses Google CDN for fonts and Bootstrap Grid system.

For Hybrid Apps

Download the zip of the latest version above.

<link rel="stylesheet" href="path/to/03/dist/library.native.min.css">

In the head of your html, reference the location to your library.native.min.css.

This distribution version includes the fonts natively and uses Flexbox Grid system.

If you download and package the library in your project remember to use a task runner or a preprocessor to compile any changes that are made to the files in the library structure.

    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, user-scalable=no, minimum-scale=1.0, maximum-scale=1.0">
    <title>Document</title>

    <!-- Icons -->
    <link rel="stylesheet" href="path/to/03/src/black-tie/css/black-tie.min.css">

    <!-- Library CSS -->
    <link rel="stylesheet" href="path/to/03/dist/library.min.css">
<head>
<body>
    <header>
        <!-- content -->
    </header>

    <main>
        <!-- content -->
    </main>

    <footer>
        <!-- content -->
    </footer>

    <!-- jQuery -->
    <script src="path/to/03/js/jquery.min.js"></script>
    <!-- Library JS -->
    <script src="path/to/03/js/library.js"></script>
</body>
</html>

Global Modifiers

Borders

.border .border--all

.border .border--top .border--bottom

.border .border--left .border--right

Remove all borders and border properties: .border .border--none

Layout

  • .padding-# - add padding in em units (1 through 10)
  • .padding-top-# - add padding to top only
  • .padding-bottom-# - add padding to bottom only
  • .padding-left-# - add padding to left only
  • .padding-right-# - add padding to right only
  • .no-padding - remove all padding
  • .margin-# - add margin in em units (1 through 10)
  • .margin-top-# - add margin to top only
  • .margin-bottom-# - add margin to bottom only
  • .margin-left-# - add margin to left only
  • .margin-right-# - add margin to right only
  • .no-margin - remove all margins

Appearance

  • .fixed - position: fixed; top: 0;
  • .center - margin: 0 auto;
  • .opacity-low - opacity: .5;
  • .rounded-corners - border-radius: 15%;
  • .rounded - border-radius: 100%;
  • .display-inline - display: inline-block;
  • .display-block - disaplay: block;
  • .block-mobile - width 100% amd force block at breakpoint small
  • .hidden - display: none;
  • .no-wrap - white-space: nowrap;
  • .overflow-hidden - overflow: hidden;

Visibility

  • .hidden-large - breakpoint large = display: none; | breakpoint huge = display: inline-block;
  • .hidden-medium - breakpoint medium = display: none; | breakpoint large = display: inline-block;
  • .hide-small - display: none; | breakpoint medium = display: block;
  • .hide-small-inline-block - display: none; | breakpoint medium = display: inline-block;
  • .show-small - breakpoint medium = display: none;
  • .hide-medium - display: block; | breakpoint medium = display: none; | breakpoint large = display: block;

Text

  • .center-text - text-align: center;
  • .ellipsis - @include ellipsis();
  • .ellipsis-block - @include ellipsis(); display: block;
  • .word-break - word-break: break-all;
  • .text-indent - padding-left: $gutterwidth;
